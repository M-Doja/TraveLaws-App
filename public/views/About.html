What is TravState?

The idea behind TravState is that if you are travelling or need information on another state that you are not familiar with, you can use this app to get high level overview of the laws, as well as a few basic demographics of the state you are interested in.

Currently this app is still under development even though is is live. Please do continue to check back with us as we continue to improve and add more features.
